# Controle de Manutenção - PAM

Sistema completo para controle de manutenção de equipamentos do Pronto Atendimento Municipal.

## Funcionalidades

- 📊 Dashboard com alertas em tempo real
- 🛠️ Cadastro de equipamentos
- 🔧 Registro de manutenções
- ⚠️ Alertas automáticos
- 📋 Relatórios quinzenais
- 📜 Histórico de alterações
- 💰 Controle de custos

## Tecnologias

- **Backend:** Python Flask + PostgreSQL
- **Frontend:** HTML5 + CSS3 + JavaScript
- **Autenticação:** JWT
- **Hospedagem:** Render.com
